📅 Dayli 개인정보처리방침

silverwest(이하 "개발자")는 Dayli(이하 "앱")을 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 아래와 같이 개인정보처리방침을 안내드립니다.

1. 수집하는 개인정보

Dayli는 사용자가 입력하는 D-Day, To-Do 등 일정 데이터를 개발자 서버로 전송하거나 수집하지 않습니다. 해당 데이터는 사용자의 기기(스마트폰)에만 저장됩니다.

단, AI 자동 입력 기능 사용 시 사용자가 직접 제공한 이미지 또는 텍스트가 Google Gemini API 서버로 전송될 수 있습니다. 이 데이터는 일정 파싱 목적으로만 사용되며, 앱은 해당 데이터를 별도로 저장하지 않습니다.

앱 내 광고 제공을 위해 Google AdMob SDK가 광고 식별자(AD_ID), 기기 정보, 광고 상호작용 정보 등을 수집하거나 처리할 수 있습니다.

백업/내보내기 기능 사용 시 데이터는 사용자가 직접 선택한 위치에 JSON 파일로 저장됩니다. 앱은 백업 파일의 저장 위치나 내용을 별도 서버로 전송하지 않습니다.

2. 개인정보의 이용 목적

AI 자동 입력 기능에 한하여, 사용자가 직접 제공한 이미지 또는 텍스트를 일정 정보 자동 파싱 및 등록에 이용합니다. 그 외 목적으로는 사용하지 않습니다.

3. 개인정보의 제3자 제공

AI 자동 입력 기능 사용 시 Google Gemini API로 이미지 또는 텍스트가 전송될 수 있습니다.

광고 제공을 위해 Google AdMob SDK가 광고 식별자(AD_ID), 기기 정보, 광고 상호작용 정보 등을 수집하거나 처리할 수 있습니다.

그 외 개발자가 직접 개인정보를 제3자에게 판매하거나 제공하지 않습니다.

· Google AI 이용약관: https://ai.google.dev/terms

4. 사용 중인 외부 서비스

Google Gemini API — 이미지/텍스트 자동 파싱 (외부 서버 전송)

Google ML Kit — OCR 텍스트 인식 (온디바이스 처리, 외부 전송 없음)

Google AdMob — 앱 내 배너 광고 제공

5. 개인정보의 보관 및 파기

모든 데이터는 사용자의 기기에만 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

AI 자동 입력 기능을 통해 전송된 이미지/텍스트는 앱이 별도로 저장하지 않습니다.

6. 광고 및 분석 도구

Dayli는 화면 하단에 배너 광고를 표시하기 위해 Google AdMob 광고 SDK를 사용합니다.

Google AdMob은 맞춤형 광고 제공, 광고 성과 측정, 부정 사용 방지를 위해 광고 식별자(AD_ID), 기기 정보, 광고 상호작용 정보 등을 수집하거나 처리할 수 있습니다.

광고 식별자는 기기 설정에서 재설정하거나 맞춤형 광고 사용을 제한할 수 있습니다.

Google의 광고 관련 정책은 https://policies.google.com/technologies/ads에서 확인할 수 있습니다.

Dayli는 별도의 자체 분석 도구를 사용하지 않습니다.

7. 아동의 개인정보

Dayli는 만 14세 미만 아동의 개인정보를 별도로 수집하지 않으며, 앱 자체가 개인정보를 수집하지 않습니다.
