📅 Dayli 개인정보처리방침

silverwest(이하 "개발자")는 Dayli(이하 "앱")을 운영함에 있어 이용자의 개인정보를 중요하게 생각하며, 아래와 같이 개인정보처리방침을 안내드립니다.

1. 수집하는 개인정보

Dayli는 기본적으로 개인정보를 수집하지 않습니다.

앱에서 사용자가 입력하는 D-Day, To-Do 등 모든 데이터는 사용자의 기기(스마트폰)에만 저장됩니다.

단, AI 자동 입력 기능 사용 시 사용자가 직접 제공한 이미지 또는 텍스트가 Google Gemini API 서버로 전송될 수 있습니다. 이 데이터는 일정 파싱 목적으로만 사용되며, 앱은 해당 데이터를 별도로 저장하지 않습니다.

2. 개인정보의 이용 목적

AI 자동 입력 기능에 한하여, 사용자가 직접 제공한 이미지 또는 텍스트를 일정 정보 자동 파싱 및 등록에 이용합니다. 그 외 목적으로는 사용하지 않습니다.

3. 개인정보의 제3자 제공

AI 자동 입력 기능 사용 시 Google Gemini API로 데이터가 전송됩니다. 그 외 어떠한 제3자에게도 개인정보를 제공하지 않습니다.

· Google AI 이용약관: https://ai.google.dev/terms

4. 사용 중인 외부 서비스

Google Gemini API — 이미지/텍스트 자동 파싱 (외부 서버 전송)

Google ML Kit — OCR 텍스트 인식 (온디바이스 처리, 외부 전송 없음)

5. 개인정보의 보관 및 파기

모든 데이터는 사용자의 기기에만 저장됩니다. 앱을 삭제하면 모든 데이터가 함께 삭제됩니다.

AI 자동 입력 기능을 통해 전송된 이미지/텍스트는 앱이 별도로 저장하지 않습니다.

6. 광고 및 분석 도구

현재 Dayli는 광고 SDK 또는 분석 도구를 사용하지 않습니다. 추후 광고가 추가될 경우 본 방침을 업데이트하겠습니다.

7. 아동의 개인정보

Dayli는 만 14세 미만 아동의 개인정보를 별도로 수집하지 않으며, 앱 자체가 개인정보를 수집하지 않습니다.

8. 개인정보처리방침의 변경

본 방침이 변경될 경우 앱 업데이트 또는 본 페이지를 통해 안내드리겠습니다.
